# React + TypeScript + Vite - Bolilerplate Pack

![Thumbnail Image](./public/thumbnail.png)

## Setup includes:

- React 19
- TailwindCSS v4
- Shadcn UI
- Vite + Typescript

## Commands

In order to run the code follow below steps:

```
git clone https://github.com/Nikhil1602/vite-boilerplate.git
cd vite-boilerplate
npm i
npm run dev
```
